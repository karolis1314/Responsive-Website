Flexbox

 As a developer you must understand in which way you want to develop the software. What way you will build the window or store information. On the website, we know as a developer that we store everything inside the box which is inside another box which is inside another box and so on. Flexbox is a CSS property but more like a tool in this case. This tool is most impowering tool you can use on the web design. It opens so many more options and pick you can choose from. It really puts your design to a professional level.Basically, everything you put on the web page are block, typo of boxes. It looks like we have three boxes inside a bigger box. Div, take up the whole width of the page as a default, they will be stacked as well from the top to the bottom of the page. So, the first element like A will be take the whole width of the container or the page, until you add a new box(div) which will go underneath in this example its B.

 We can change this type of behaviour by coding the function to, place it in line not in the block, so it that case the block would only take as much space as content inside of them. This is a basic layout concept. It is not enough for complex web development; therefore, we have flex box. When we have flex what we are doing is defining the relationship between the parent box and the child box. Parent box is the box that contains all the other boxes and the child which is the content in the boxes.All the properties can be changed within the child boxes. Make sure they are cantered, or vertical and so on. Any layout you can see on the websites, it is usually spacing out or aligned with flex box. This tool is all you need for layouts or distribution of information in your website. It makes the website look very professional and clean. Flex box is very important cause it allows us as developers to make responsive websites much simpler and faster.
