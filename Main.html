Home Page Introduction

  On this assignment we had to make a responsive website, that would use grid and flex box attributes. Website must be informative and detailed on the grid and flex attributes. Website must be responsive. In this case the website will be responsive to large screens, tablet, mobile phone. Examples were taken using iphone6 screen on the web engine and iPad. On this website I focused to make sure it looks aesthetic and clean with some flavour of animation to the navigation bar. My main goal was to understand and implement responsive website design with flex box and CSS-grid. There were some issues I have ran into, but I will discuss them later. Responsive web design is a must now a days, all the websites are mainly accessed by users through out mobile device or some sort of a tablet. Laptops and desktop pcs are not as common anymore. Mobile apps and web browsing have taken over the 21st century. Everything is done online now a days, more and more things are moving to web servers and websites.

  Even gaming one day will be on the website without a need of a powerful gaming machine. You will be able to rent a device somewhere in the world, which will run your games and you will access it through the web browser on a website. In this matter responsive website are a must. Understanding how it is responsive, how can it be done better or changed, how the layouts work, what does what is a must for a software engineer. For any kind of job in software industry you must have at least a basic understanding of website functionality and responses. As I have mentioned before, everything will move on, on the website from games to virtual reality. This project gave me a huge understanding on these three simple functionalities which are so crucial. I got a wider understanding on CSS and HTML documents.
