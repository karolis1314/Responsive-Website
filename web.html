Responsive Web Design

 Responsive web design is the most important function in todayâ€™s web development. Understanding that the most websites are accessed through a mobile device instead of a large screen is crucial. Most of the users today, are using mobiles, apps, or web browsing. Responsive web design is a solution to this problem of users using, multiple devices with different screen sizes or resolution. It removes the old school browsing of pinching and zooming the on the website.

 Flow HTML design is the key for responsive features. As the browser gets smaller all the items will drop vertically on the website which takes less space. We no longer need to use px as the measurements on our website design sheets. Responsive web allows us to use such as %, vh, vw, em, rem measurements which are way more accurate and proportional to the whole website, especially when it must work as a responsive on. This allows us to have breakpoints too, which are screen sizes or web sizes that changes the layout on the size you choose or view the website in. We can nest objects and content inside a div which is a container and manipulate it as we please on the website. There is a huge debate in the industry which device should be designed first, mobile or desktop, I found in this assignment that it is easier to design basic parameters without the screen size, then go from mobile to desktop. I found it that it gives most room to manipulate the screens and add any other size easily. I think in the future I will rotate my view and start from the mobile device and work my way up, due to the fact that most of the users are using mobile devices much more then desktop or any other larger screens.
